Variables

Variables

Variables are containers which hold data.

They have no value until one is assigned to it.

A variable must have a unique name.

We can refer to a variable to retrieve its value.

Declaring Variables

When we declare a variable, we preface the variable name it with var.

Once a variable has been created, we do not need to include var in future references.

Examples

var myCat;
var myBoyCat = "Flynn";
var myGirlCat = "Quorra";
myCat;      // empty!
myBoyCat;   // "Flynn"
myGirlCat;  // "Quorra"

Naming Variables

  • must start with a letter, dollar sign ($) or underscore (_)
  • may not contain a dash (-) or period (.)
  • are case sensitive
  • should be descriptive
  • should be written in camelCase * (var lowerCapsCaps;)
  • may not contain keywords or reserved words

* underscores between words are acceptable, but camelCase is preferred JavaScript style

Constants

If we know a variable's value will not (and/or should not change), we can identify it as a constant.

var PI = 3.14;

Here's the catch; ES6 introduces a formal way to declare a constant (const). A browser you code for may not yet recognize ES6. In the meantime, break the camelCase convention; write out constants using UPPERCASE characters. The language will not enforce your constant, but you will know the 'variable' should not be altered.
