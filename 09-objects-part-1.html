Objects, part 1

Objects

An object is a collection of properties and methods.

Properties

A property describes a characteristic of an object and stores a value. Property values may be any data type.

You may hear the term "key" and "value" to describe the components of a property.

Methods

A method is an action performed on a data type.

We will break down methods in detail in the next section. After we learn functions control flow and loops, we will be able to create our own methods!

How can we describe a cat?

Cats

A cat has the following properties:

  • species
  • name
  • age
  • a long or short coat
  • four paws
  • different colors
  • personality
    • shy?
    • friendly?
    • bites?

Let's describe Quorra

var quorra = {
  species: "cat",
  age: 5,
  paws: 4,
  coat: "short",
  colors: ["brown", "gray", "black", "white"],
  shy: true,
  friendly: true,
  bites: false
}

Creating an Object

An array can be created using an object literal or an object constructor.

Object Literal

An object literal uses curly brackets ({ }) to enclose the key-value pairs. Each pair is separated by a comma. The last value does not require a comma.

var car = {
  make: "Chevrolet",
  model: "Camaro",
  year: 2016
}

Key-value pairs may be on one line, but readability improves by placing each on an individual line.

Object Constructor

Values of the object are provided as arguments to a new Object() object. Each key is then appended to the object and the value assigned.

var car = new Object();

car.make = "Chevrolet";
car.model = "Camaro";
car.year = 2016;

car; // {make: "Chevrolet", model: "Camaro", year: 2016}

Accessing Object Properties

Objects are not indexed.

We access an object's property values by using bracket or dot notation.

Bracket Notation

car["make"];   // "Chevrolet"
car["model"];  // "Camaro"
car["year"];   // 2016

This should look familiar from working with arrays. Instead of using an index value, we refer to the property's 'key' (which should be enclosed in quotes).

Dot Notation

car["make"];   // "Chevrolet"
car["model"];  // "Camaro"
car["year"];   // 2016

Bracket or Dot?

As you are learning, practice both.

After we address methods, you may develop logic behind when to use one and when to use the other.

Changing Property Values

To change a property's value, simply re-assign it a new value (as you would a variable).

car["make"] = "Dodge";
car.model = "Challenger";

Deleting Properties

To delete a property:

delete car.make
