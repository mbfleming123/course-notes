Console Basics

JavaScript Console

Modern browsers include a command-line interface which lets us work step-by-step with JavaScript.

To open the JavaScript Console in Google Chrome:

  • Open a blank tab
    • Type about:blank in the URL bar
  • View → Developer → JavaScript Console or
  • ☰ (top right corner) → More Tools → JavaScript Console or
  • Keystroke: Option-Command-J

REPL

The JavaScript Console is the example of a REPL

  • Reads your command
  • Evaluates the statement or expression
  • Prints a result
  • Loops ... repeats the previous steps!

Using the Console

  • tinker with very simple statements
  • interact with object models
    • document
    • window
  • be alerted to errors when code is incorrect
  • isolate parts of a script loaded into the browser

Let's Play!

Don't worry if the commands we type do not yet make sense.

Let's write a few statements to get familiar with the console.

Write the command in your notebook (and what it does), then type it into the console to see it work.

Write to the console

console.log("Hello")

Writes "Hello" to the console.

Write to the DOM

document.write("Hello")

Writes "Hello" to the Document Object Model (our web page).

Trigger an alert

window.alert("Hello")

Displays an alert box with "Hello"

Calculate

1 + 1

3 - 2

4 * 5

Displays the result of an expression to the console.

Calculate a square root

Math.sqrt(25)

Returns the square root of 25.

Value of Pi (Π)

Math.PI

Returns the value of Pi (3.14...)

Transform text

"hello".toUpperCase()

Transform "hello" to uppercase letters.

Replace a character

"car".replace("r", "t")

Replace the "R" in "car" with a "T".

Assign a value to a variable

var cats = 2

cats stores a value of 2.

Return the value of a variable

cats

cats returns the value of 2.

Clear the console

clear()

Give yourself a fresh start.

Tips

If you are unsure about how a basic aspect of JavaScript works, try entering it in the console like we did above.

If the example requires multiple lines of code, type shift-return at the end of each line to keep the code from executing. The cursor will move to the next line so you can continue typing your command(s).

Testing Big Scripts

Sometimes it is easier to write code in a file than the console. Include the script in a basic HTML page, then load it into the browser.
