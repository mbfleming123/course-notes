Arrays

Arrays

An array stores an ordered list of values. It may contain any data type.

["cat", "dog", "fish"]

[35, 16, 2, 40]

[true, false, false]

["cat", 3, true]

Creating Arrays

An array can be created using an array literal or an array constructor.

Array Literal

Values are placed within brackets ([ ]) and separated by commas. The last value does not require a comma.

var lieDetector = [true, false, false];

Values may also be placed on individual lines.

var lieDetector = [
  true,
  false,
  false
];

Array Constructor

Values of the array are provided as arguments to a new Array() object.

var petStore = new Array("cats", "dogs", "fish");

As with the array literal, values may be placed on individual lines.

var petStore = new Array(
  "cats",
  "dogs",
  "fish"
)

If no values are provided to Array(), an empty array will be created.

var array = new Array();
array;  // []

Array Index

Arrays are indexed. The index begins at zero (0).

["cats", "dogs", "fish"]

array item index value
cats 0
dogs 1
fish 2

Array Length

Since arrays are indexed, we can determine the size of an array with length.

var petStore = ["cats", "dogs", "fish"]
petStore.length;  // 3

Accessing Array Items

We can access individual items in an array using bracket notation.

var petStore = ["cats", "dogs", "fish"]
petStore[0];  // cats
petStore[1];  // dogs
petStore[2];  // fish

Adding Array Items

We can also add to an array by specifying the index in which to save a value and assigning the value.

var array = [];
array[0] = "Hello";
array[1] = "world";
array; // ["Hello", "world"]

The last item

JavaScript does not provide an elegant way to return the last item of an array using bracket notation. (Some languages actually do!)
