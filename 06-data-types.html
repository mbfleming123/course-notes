Data Types

Data Types

Data types are the most basic 'building blocks' which can be used in programming.

  • boolean
  • number
  • string
  • null
  • undefined
  • symbol (ES6)

Booleans

There are only two Boolean values:

  • true
  • false

Booleans are commonly used in conditional tests. (i.e. "If true, do X; if false, do Y")

Numbers

If made it this far in academia, you should be familiar with these. :}

  • integers (1, 2, 3)
  • floating-point numbers (a.k.a. "floats") (1.0, 2.0, 3.0)
  • special numbers (Infinity, -Infinity, NaN)
    • NaN stands for "Not a Number"

Numbers are valuable (har, har!) for calculations, comparisons, counting, date/time and positioning.

Strings

Strings are a combination of characters which represent text or code.

Strings which represent text are enclosed in single or double quotes. (i.e. "string")

Strings may be concatenated (linked together). (i.e. "Hello " + "world!")

null & undefined

null is an object which has no value on purpose. It is not 'false' or empty.

var dev = null;
dev = false;
dev = "";

Something is undefined when it contains no value or simply does not exist.

var cats;
cats;
dogs;
